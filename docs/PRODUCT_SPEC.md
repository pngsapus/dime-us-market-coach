# Product Spec

## V1 Scope

- Dashboard with market summary and data freshness.
- Radar using mock symbols `NVDA`, `AMD`, and `TSLA`.
- Stock Explain in polished beginner-friendly Thai.
- Practice Plan with entry zone, stop loss, take profit, Risk:Reward, expected loss, expected profit, and risk profile pass/fail.
- Dime Check where the user manually enters the real price seen in Dime.
- Journal for decisions, reasons, results, and lessons.
- Settings / Risk Profile.
- Data Status page for provider health and limitations.

## Out of Scope

- Auto trading.
- Dime API connection.
- Real market data provider.
- LINE Official Account.
- Predictions of future stock price.
- Real-time alerts.
